1. Project Overview

This project is a simple console-based Calculator created using Python.
It performs basic arithmetic operations such as addition, subtraction, multiplication, and division.
The purpose of this project is to demonstrate the use of VS Code, Git, and GitHub for software development, version control, and documentation.

2. Project Objective

To understand how to build a small functional program using Python

To learn how to use VS Code for coding and testing

To practice Git commands such as git init, git add, git commit, and git push

To maintain a clean repository structure on GitHub

To create proper documentation for a mini-project

3. Tools and Technologies Used

Programming Language: Python

Code Editor: Visual Studio Code

Version Control: Git

Repository Hosting: GitHub

4. Development Process
Step 1: Project Setup

Created a folder for the project

Opened it in VS Code

Created the main Python file named calculator.py

Step 2: Writing the Program

Added functions for each arithmetic operation

Added input handling

Added checks for invalid operations and division by zero

Tested the program in VS Code terminal

Step 3: Initializing Git

Executed these commands in VS Code terminal:

git init
git add .
git commit -m "Initial commit"

Step 4: Making Multiple Commits

Made meaningful commits for changes in code, README file, documentation, and screenshots.

Step 5: Pushing to GitHub

Created a repository on GitHub and pushed the project using:

git remote add origin <repo-link>
git branch -M main
git push -u origin main

Step 6: Adding Documentation

Created a docs/ folder and added this documentation file.
